## 1. Приобретение домена

Для начала вам требуется свой домен.
- [Где приобрести домен?]({{ links.domainBuyUrl }} "blank")

## 2. Выбор DNS серверов

Вы можете использовать DNS сервера сервисов по регистрации доменов, но мы рекомендуем [CloudFlare]({{ links.cloudflareUrl }} "blank"). На его примере и разберем как сделать подключение.

## 3. Создание A записи

После привязки домена к DNS серверам Cloudflare, нам нужно создать A запись, она указывает к какому IP адресу обращаться клиенту.

---

Параметры:
- Name: ваш основной домен.
- IPv4 address: IP адрес (Он есть на вкладке 'Сеть' в панели управления сервером.)

<Figure>
    ![](/assets/minecraft/domain/a-record.jpg)
</Figure>

## 4. Создание SRV записи

---

Параметры:
- Name: Ваш домен
- Protocol: TCP
- TTL: auto (если нет, то 2 минуты)
- Priority: 1
- Weight: 1
- Port: порт вашего сервера
- Target: домен с поддоменом из A записи

<Figure>
    ![](/assets/minecraft/domain/srv-record.jpg)
</Figure>

## 5. Все готово! Можно заходить на сервер по адресу server.ваш_домен.ru
