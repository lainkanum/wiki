# Как установить свой ресурспак на сервер?

## 1. Загрузите желаемый ресурспак на данный ресурс

- [Загрузить ресурспак]({{ links.ResourceUploaderUrl }} "blank")

<Figure>
    ![](/assets/minecraft/server-resourcepack/resource-upload.png)
</Figure>

<ColorBlock color="red">
- На данном ресурсе файл не может превышать 95 МБ
</ColorBlock>

## 2. Откройте файл `server.properties`.
Вставьте в строку `resource-pack=` ссылку из блока `Download URL`;

В строку `resource-pack-sha1=` - ссылку из блока `SHA-1 Hash`.

<Figure>
    ![](/assets/minecraft/server-resourcepack/resource-configure.png)
</Figure>

## 3. Перезапустите сервер


<ColorBlock color="yellow">
- Вместо представленного ресурса можно использовать любой другой файлообменник с возможностью копирования прямой ссылки на загруженный файл, однако, вам потребуется вручную рассчитать SHA-1 хэш.
</ColorBlock>
