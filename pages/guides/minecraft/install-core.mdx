# Как сменить ядро/версию сервера Minecraft?

## 1. Выключите сервер и удалите все папки в вкладке "Файлы":

<ColorBlock color="red">
    - Все данные сервера полностью удалятся, если они вам нужны - сохраните, прежде чем удалять.
</ColorBlock>

<Figure>
    ![](/assets/minecraft/install-core/delete-files.png)
</Figure>

## 2. Скачайте нужное вам ядро желаемой версии:

 - [Выбрать ядро можно здесь](/resources/minecraft/cores)

## 3. Перенесите файл ядра в файлы сервера:

<Figure>
    ![](/assets/minecraft/install-core/install-core.png)
</Figure>

## 4. Переименуйте файл ядра на `server.jar`:

<Figure>
    ![](/assets/minecraft/install-core/rename-core-1.png)
</Figure>

<Figure>
    ![](/assets/minecraft/install-core/rename-core-2.png)
</Figure>

## 5. Поменяйте название ядра в команде запуска во вкладке "Настройки" на `server.jar`:

<Figure>
    ![](/assets/minecraft/install-core/change-startup-1.png)
</Figure>

<Figure>
    ![](/assets/minecraft/install-core/change-startup-2.png)
</Figure>

## 6. Смените версию Java на соответствующую для ядра:

<Figure>
    ![](/assets/minecraft/install-core/change-java.png)
</Figure>

## 7. Запустите сервер и установите `eula=true`

После запуска сервера, он выключится - это нормально, потому что мы удалили все файлы.
Дабы он работал, а не выключался, нам нужно принять пользовательское соглашения "EULA".

Зайдите в вкладку "Файлы", найдите файл `eula.txt`, там будет строка `eula=false`, нужно установить `eula=true`.

<Figure>
    ![](/assets/minecraft/install-core/accept-eula.png)
</Figure>

## 8. Все готово! После запуска сервера, он будет работать на том ядре и версии Minecraft, которую вы установили.
