### Paper
Высокопроизводительное серверное ядро нового поколения. Предлагает максимальную оптимизацию серверной части, улучшенную защиту от эксплойтов, автоматическую подгрузку чанков и значительно меньшее потребление ресурсов.

> Версии: {{ versions.paper }}

<ColorBlock color="green">
Рекомендуемое ядро для обычной игры.
</ColorBlock>

[Скачать]({{ links.paperUrl }} "blank")

---

### Forge
Самая популярная модданг-платформа с огромной библиотекой модов. Отличается простотой установки модификаций, поддержкой большинства старых и новых модов, а также мощным API для разработчиков.

> Версии: {{ versions.forge }}

[Скачать]({{ links.forgeUrl }} "blank")

---

### NeoForge
Современная модданг-платформа нового поколения, созданная как прогрессивная альтернатива классическому Forge. Отличается лучшей оптимизацией, расширенным API для разработчиков, удобством установки модов, а также активной поддержкой актуальных версий Minecraft с фокусом на будущее модинга.

> Версии: {{ versions.neoForge }}

[Скачать]({{ links.neoForgeUrl }} "blank")

---

### Fabric
Легковесная модданг-платформа нового поколения. Выделяется минимальным оверхедом, быстрой работой, современным подходом к моддингу и существенно меньшим потреблением ресурсов по сравнению с аналогами.

> Версии: {{ versions.fabric }}

[Скачать]({{ links.fabricUrl }} "blank")

---

### Pufferfish
Оптимизированное ядро Minecraft на основе Paper с улучшенной производительностью, снижением нагрузки на сервер и поддержкой плагинов.

> Версии: {{ versions.pufferfish }}

[Скачать]({{ links.pufferfishUrl }} "blank")

---

### Purpur

Высокооптимизированное ядро с множеством настроек. Предоставляет расширенные возможности кастомизации сервера, улучшенную производительность и гибкость в настройках.

> Версии: {{ versions.purpur }}

[Скачать]({{ links.purpurUrl }} "blank")

---

### Mohist

Уникальное ядро, поддерживающее одновременно моды и плагины на базе Forge. Идеально подходит для серверов, которые хотят использовать функционал как модификаций, так и плагинов без потери производительности.

> Версии: {{ versions.mohist }}
